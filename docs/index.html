InputDetect

概要

InputDetect は、マウス・タッチ（マルチタッチ対応）など多様な入力デバイスの操作をシンプルに検知・処理できる JavaScript ライブラリです。 ドラッグ、ボタン押下、座標検知、ピンチイン・アウト（2本指操作）、ホイール操作など、Webアプリケーションでの複雑な入力処理を統一的に扱うことができます。

詳しい説明は、Github Pagesで解説しています。

特長

  • マウス・タッチ入力の抽象化
  • 左・右・中央ボタンやドラッグ操作、ホイール操作に対応
  • 3本指までのマルチタッチを検知
  • イベントリスナーを簡単に設定可能
