IDDraggableSwitch

InputDetect. IDDraggableSwitch

ドラッグ可能なスイッチ（ボタン）の状態を管理するクラスです。 クリックやドラッグ操作の開始・終了・移動を追跡し、イベントごとに内部状態を更新できます。
